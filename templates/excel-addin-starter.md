# Excel Add-in Starter

A planned Office.js starter for Wisesheets-powered spreadsheet workflows.

## Best For

- Excel-native financial analysis
- Analyst workflows
- Spreadsheet dashboards
- AI-assisted modeling

## Wisesheets Endpoints

- `GET /v1/companies?q=<query>`
- `GET /v1/financials/`
- `GET /v1/statements/:ticker`

## Suggested Features

- Pull financial metrics into worksheets
- Search tickers from a task pane
- Refresh selected ranges
- Generate financial statement tabs
- Export audit metadata and source tags
