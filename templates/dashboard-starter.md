# Dashboard Starter

A React dashboard for company fundamentals, five-year trends, ticker search, financial statements, and health scoring.

## Best For

- Investor dashboards
- Fundamentals terminals
- Company overview pages
- AI-agent customization demos

## Stack

- React
- Vite
- TypeScript
- Tailwind CSS
- Recharts

## Wisesheets Endpoints

- `GET /v1/companies?q=<query>`
- `GET /v1/financials/`
- `GET /v1/statements/:ticker`
