User workflow

From template to custom app

  1. Pick a recipe.Choose the starter closest to the app you want to build.
  2. Use the GitHub template.Create your own repo copy without forking.
  3. Add your API key.Set WISESHEETS_API_KEY in .env or runtime secrets.
  4. Open with an AI agent.Ask it to read README.md, AGENTS.md, CUSTOMIZE.md, and API docs.
  5. Customize and deploy.Keep API calls server-side through the provided proxy.
