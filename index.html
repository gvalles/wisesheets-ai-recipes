User workflow

From recipe folder to custom app

  1. Pick a folder.Start with fundamental-stock-dashboard/, the source for the live demo.
  2. Get a free key.Create a Wisesheets API key at wisesheets.io/api.
  3. Add your key.Set WISESHEETS_API_KEY in the recipe folder's .env.
  4. Open with an AI agent.Ask it to read the folder README, AGENTS.md, CUSTOMIZE.md, and the official API docs.
  5. Customize and deploy.Keep API calls server-side through the provided proxy.
